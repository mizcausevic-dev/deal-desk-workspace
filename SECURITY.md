# Security Policy

## Reporting a Vulnerability

If you discover a security issue in this repository, please do not open a public issue with exploit details.

- Contact: [Mirza Causevic on LinkedIn](https://www.linkedin.com/in/mirzacausevic)
- Include: affected area, reproduction details, impact, and suggested remediation if available

## Scope

This repository is a portfolio project demonstrating frontend systems design, pricing governance workflows, and operator-facing deal desk interfaces. It is not intended for production deployment without authentication, data governance, access controls, and deeper commercial workflow hardening.
